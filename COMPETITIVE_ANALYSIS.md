# 📖 Introduction

Uber's Clap évolue dans un marché où les chauffeurs professionnels utilisent actuellement plusieurs types d'outils :

- applications de réservation
- agendas classiques
- outils de facturation
- CRM
- tableurs
- applications de gestion

L'objectif est d'identifier les limites existantes afin de proposer une solution plus adaptée aux chauffeurs VTC indépendants.

---

# 🎯 Objectif de l'analyse

Comprendre :

- les solutions déjà utilisées
- leurs points forts
- leurs limites
- les opportunités de différenciation
